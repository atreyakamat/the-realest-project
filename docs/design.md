# Design System

## Visual Direction
EstateFlow CRM uses a premium dark, mobile-first interface with calm contrast, clear hierarchy, and action-heavy cards.

## Layout Principles
- Bottom navigation for primary routes.
- Large stacked cards on mobile, denser grids on desktop.
- Sticky action areas for frequent CRM tasks.
- Fast scanning with badges, chips, and short metadata lines.

## Components
- Metric cards for dashboard reporting.
- Lead cards with quick actions.
- Property cards with inventory details and status.
- Timeline rows for activity history.
- Forms with large inputs, clear feedback, and inline status messages.

## Interaction Rules
- Favor one-tap actions over multi-step flows.
- Use optimistic feedback and readable loading states.
- Keep destructive actions explicit and uncommon.
- Support touch-first usage without sacrificing desktop clarity.
