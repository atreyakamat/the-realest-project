# Tech Stack

## Frontend
- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui style components

## Backend
- Next.js route handlers
- Server Actions
- Zod validation

## Data and Auth
- Supabase Postgres
- Supabase Auth
- Supabase Storage
- Supabase Realtime where useful

## Integrations
- Twilio Voice for call bridging
- Twilio WhatsApp/SMS or WhatsApp Cloud API adapter
- Resend or SMTP for email
- Browser geolocation for attendance
- OpenAI-compatible adapter for drafting assistance

## Deployment
- Vercel for hosting
- Supabase for database, auth, and storage
